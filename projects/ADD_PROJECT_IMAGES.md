## How to Add Images

1. **Take screenshots** of your projects
2. **Resize** them to approximately 800x600px (or similar landscape ratio)
3. **Save as JPG or PNG**
4. **Place in** `projects/` folder with these exact names:
   - `rail-madad.jpg`
   - `tongue-disease.jpg`

## Image Tips

- Use high-quality screenshots
- Show the main interface/feature
- Keep them clear and professional
- Landscape format works best
- Optimize file size (under 500KB if possible)
